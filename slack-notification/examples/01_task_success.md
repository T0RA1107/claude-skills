# Example: Successful Task Completion
Use this format for long-running batch jobs or data processing scripts.

```bash
scripts/slack-notify.sh "✅ *SUCCESS: Data Processing*
*Project*: [Project Name]
*Duration*: [HH:mm:ss]
*Summary*: The background data processing task has completed successfully. All records were validated and stored.
*Metrics/Highlights*:
• Records Processed: 1,250
• Success Rate: 100%
• Data Integrity: Verified
*Next Action*: I recommend reviewing the output files in the \`./results/\` directory."
```
